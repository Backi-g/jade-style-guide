Botones

Los botones deben de ser implementados de acuerdo a la especificación de bootstrap 4 en:

  • Tamaño
  • Estado
  • Estilo
  • Tipo

Tamaño

  • Largo: utilizar la clase .btn-lg
  • Mediano: utilizar la clase .btn-md
  • Pequeño: utilizar la clase .btn-sm

Código

<button class="btn btn-primary btn-lg" >Largo</button>
<button class="btn btn-primary btn-md" >Mediano</button>
<button class="btn btn-primary btn-sm" >Pequeño</button>

Estado

  • Activo: Se puede utilizar el tag button o la pseudo clase active
  • Inactivo: Atributo disabled

Código

<button class="btn btn-primary btn-lg" >Activo</button>
<button class="btn btn-primary btn-md" disabled >Inactivo</button>

Estilo

  • Primario: utilizar la clase .btn-primary
  • Secundario: utilizar la clase .btn-secondary
  • Success: utilizar la clase .btn-success
  • Información: utilizar la clase .btn-info
  • Warning: utilizar la clase .btn-warning

Código

<button class="btn btn-primary" >Primario</button>
<button class="btn btn-secondary" >Secundario</button>
<button class="btn btn-success" >Success</button>
<button class="btn btn-info" >Información</button>
<button class="btn btn-warning" >Warning</button>

Tipo

  • Submit: utilizar el value como submit en tags que no son button, si el tag es button usar type='submit'
  • Reset: utilizar el value como reset en tags que no son button, si el tag es button usar type='reset'

Código

<button class="btn btn-primary" >Primario</button>
<button class="btn btn-secondary" >Secundario</button>

Formularios

En esta sección se muestran los elementos básicos que se utilizarán en los formularios de la aplicación

  • Inputs
  • Checkboxes / Radios
  • Dropdowns
  • Validaciones

Inputs

Bootstrap utiliza HTLM 5, por lo tanto todos los elementos requieren un atributo type

Tipos de inputs:

text, password, datetime-local, date, month, time, week, number, email, url, search, tel, color

Agregue las clases .form-control-lg .form-control-md y .form-control-sm para determinar el ancho del input

Código

<div class="form-group">
  <label for="usr">Name:</label>
  <input type="text" class="form-control" id="usr">
</div>
<div class="form-group">
  <label for="pwd">Password:</label>
  <input type="password" class="form-control" id="pwd">
</div>

Checkboxes

Permite al usuario elegir más de una opción

Agregue la clase .checkbox-inline para que los checkboxes aparezcan en la misma linea

Agregue la clase .form-check para que los checkboxes y radio buttons tengan el espaciado correcto

Código

<div class="form-check">
  <label class="form-check-label">
    <input class="form-check-input" type="checkbox" value="">Option 1 (Active)
  </label>
</div>
<div class="form-check disabled">
  <label class="form-check-label">
    <input class="form-check-input" type="checkbox" value="" disabled>Option 2 (Disabled)
  </label>
</div>

Radio Buttons

Limita al usuario a elegir una sola opción

Agregue la clase .form-check para que los checkboxes y radio buttons tengan el espaciado correcto

Agregue la clase .disabled para que los checkboxes y radio buttons estén deshabilitados

Código

<div class="form-check">
  <label class="form-check-label">
    <input class="form-check-input" type="radio" value="" name="radios" > Option 1 (Active)
  </label>
</div>
<div class="form-check disabled">
  <label class="form-check-label">
    <input class="form-check-input" type="radio" name="radios" value="" disabled>Option 2 (Disabled)
  </label>
</div>

Dropdowns

Permite al usuario elegir una opción de una lista predefinida

Agregue la clase .dropdown-toggle-split para agregar espacio entre el texto del dropdown y el icono

Agregue las clases .btn-lg, .btn-md y .btn-sm en el botón para determinar el tamaño del dropdown

Agregue un header al dropdown poniendole la clase .dropdown-header

Opción 1 Opción 2 Opción 3

Código

< div class="dropdown">
  < button class="btn btn-secondary dropdown-toggle" type="button" id="dropdownMenuButton" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
   Selecione una opción
  </button>
  <div class="dropdown-menu" aria-labelledby="dropdownMenuButton">
   <a class="dropdown-item" href="#">Opción 1</a>
   <a class="dropdown-item" href="#">Opción 2</a>
   <a class="dropdown-item" href="#">Opción 3</a>
  </div>
</div>

Validaciones

Bootstrap contiene estilos de validaciones para casi todos los componentes

Agregue las siguientes clases en los input que desea validar

  • .is-valid
  • .is-invalid

Utilice las clases .valid-feedback y .invalid-feedback para agregar texto que ayude al usuario

En la página del Museo del Jade, se agregará textos de retroalimentación solo en los inputs que tienen errores

Por favor ingrese una dirección válida de correo test@email.com

Código

< div class="form-group">
  < label class="form-control-label" for="inputSuccess1" >Nombre</label>
  <input type="text" class="form-control is-valid" id="inputSuccess1">
</div>
< div class="form-group">
  < label class="form-control-label" for="inputWarning1" >Email</label>
  <input type="email" class="form-control is-invalid" id="inputWarning1">
  < div class="invalid-feedback">
   Por favor ingrese una dirección válida de correo test@email.com
  </div>
</div>
